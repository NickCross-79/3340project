Admin Documentation

Reference guide for all admin features on the Lancer Loot platform.

Overview

The Admin Dashboard gives privileged users visibility and control over site content and accounts. It is divided into four areas: Themes, Listings, Users, and this documentation.

Demo note: Anyone logged in can toggle their admin status on the Profile page. This is intentional for demonstration purposes.

Theme Switcher

The platform ships with six colour themes. Selecting a theme changes the look of every page instantly.

  • Default – Clean white & purple, the standard look.
  • Playful – Warm pastels with rounded corners.
  • Winter – Cool icy blues for a seasonal feel.
  • Blue & Gold – University-branded palette.
  • Earthy – Warm browns and natural tones.
  • Nature – Greens and forest colours.

Themes are stored in localStorage under the key theme and are applied before the page renders to prevent a flash of unstyled content.

Listing Management

Admins can search, filter, edit, and delete any product listing on the platform.

  1. Use the search bar and status filter to narrow down results.
  2. Click Edit on any row to open the listing editor modal.
  3. Editable fields: Title, Description, Price, Category, Status, Condition.
  4. Click Delete to permanently remove a listing and all associated cart/order references.
Deleting a listing is irreversible. The confirmation dialog must be accepted before the deletion is processed.

User Administration

The Users panel lists every registered account with their status and role.

  • Disable Account – Prevents the user from logging in. Existing sessions are not immediately invalidated but the next login attempt will be blocked.
  • Re-enable Account – Restores login access for a previously disabled user.
  • Grant Admin – Promotes a standard user to admin, giving them access to this dashboard.
  • Revoke Admin – Removes admin privileges from a user. They will retain their account but lose dashboard access.
You cannot disable your own account through this panel.

Admin Toggle (Demo Feature)

Because this is a demonstration site, any logged-in user can grant or revoke their own admin rights from the Profile page. In a production environment this toggle would be removed and admin assignment would be controlled by a super-admin or directly in the database.

Security Notes

  • All admin API endpoints verify session-level admin status server-side (requireAdmin()).
  • Input is validated and parameterised queries are used throughout to prevent SQL injection.
  • All API responses include X-Content-Type-Options: nosniff.
  • Session cookies are HttpOnly and SameSite=Strict.
